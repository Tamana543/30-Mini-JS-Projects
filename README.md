## Why This Project

I built this collection to grow **one project at a time**. Instead of rushing into big frameworks, I focused on mastering the core — JavaScript itself.

- Practice JavaScript every day
- Turn small ideas into working features
- Strengthen logic, confidence, and creativity
- Build discipline through repetition

This repo reflects my learning mindset.

- Includes **30 independent mini projects**
- Each project explores a specific JavaScript concept
- Projects run directly in the browser
- Covers logic, DOM interaction, and UI behavior

Small projects. Real learning.
